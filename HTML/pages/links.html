Estudos de links numa pagina HTML

Links são utilizado para navegação entre páginas ou para outras partes de um documento, no caso de linkar uma outra pagina você tem o atributo href onde fica a URL e o atributo target que diz se o página sera aberta na mesma aba ou aba nova, definido como _blank uma nova aba sera aberta, exemplo utilizando CSS para estilização.
