Tag's utilizadas para estruturar texto no documento

<h1> <h2> <h3> <h4> <h5> <h6> são utilizadas para hierarquia de titulos, h1 maior h6 menor.

<p> utilizado para criação de paragrafos

<strong> utilizado para deixar o texto negrito (Importante)

<em> utilizado para deixar italico (ênfase)

<mark> utilizada pra deixar o texto marcado

<br> é utilizado para quebrar linha.

<a> define um hiperlink, onde o atributo href é o endereço para onde quer navegar.

<hr> utilizado para criar um linha horizontal
